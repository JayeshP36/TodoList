TODO List Application
Overview
The TODO List Application is a modern task management solution designed to streamline your daily workflow. Built with Spring Boot for the backend and ReactJS for the frontend, this application offers a seamless user experience coupled with powerful functionality.
Technologies Used
•	Backend:
o	Spring Boot - A powerful Java framework for building RESTful APIs.
o	IntelliJ IDEA - Integrated Development Environment (IDE) used for backend development.
o	Postman - API testing tool for verifying backend endpoints and requests.
•	Frontend:
o	ReactJS - A JavaScript library for building user interfaces.
o	Visual Studio Code (VSC) - Lightweight and powerful code editor used for frontend development.
o	Postman - Although primarily used for backend testing, Postman can also be used to test frontend APIs.
Features
•	Create Tasks: Easily create new tasks with a simple interface.
•	Update Tasks: Update task details or mark tasks as completed.
•	Delete Tasks: Remove tasks that are no longer needed.
•	Responsive Design: The frontend interface is responsive and adapts to different screen sizes for a seamless user experience.
Installation
Backend
1.	Clone the repository.
2.	Open the project in IntelliJ IDEA.
3.	Run mvn spring-boot:run to start the Spring Boot application.
Frontend
1.	Clone the repository.
2.	Open the frontend directory in Visual Studio Code.
3.	Run npm install to install dependencies.
4.	Run npm start to start the React development server.
Usage
1.	Access the TODO List application through your web browser.
2.	Begin managing your tasks by creating, updating, or deleting them as needed.
3.	Test backend APIs using Postman to ensure proper functionality and performance.
